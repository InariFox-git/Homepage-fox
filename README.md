# Homepage-fox  
Персональная стартовая страница для быстрого доступа к избранным ресурсам.

Может быть установлен локально или на удалённом сервере.\
Возможность работы с несколькими пользователями.

![ScreenShot](/static/sc.jpg)

## Установка 
1. ``` bash pip install -r requirements.txt ```
2. ``` python app.py ```
3. 127.0.0.1:5000/?user_id=Username
